Projet Python

Le projet Python a pour objectif de collecter, stocker et analyser les journaux de navigation des employés de l'entreprise STESIO. Ces logs, enregistrés dans une base de données relationnelle, permettent de suivre les accès aux sites web et d'extraire des statistiques pertinentes.

Ce projet repose sur une architecture orientée base de données et nécessite des traitements adaptés pour organiser efficacement les logs et en extraire des informations exploitables.

Objectifs

  • Importer et structurer les journaux de navigation
  • Stocker les données dans une base relationnelle
  • Fournir des analyses sous forme de statistiques et rapports
  • Répondre à des requêtes spécifiques pour assurer le suivi des accès web

Technologies utilisées

Langages

  • SQL
